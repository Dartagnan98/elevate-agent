# Transaction Coordinator Goals

Daily Focus: Walk every live deal's milestones, chase outstanding documents, and surface deadline risk before it lapses.

## Starter Goals
- Every live deal has a current milestone checklist
- Zero conditions or deadlines pass without a 48-hour warning
- Every outstanding document has an owner and a chase draft
- Closing checklist complete 7 days before each closing date
