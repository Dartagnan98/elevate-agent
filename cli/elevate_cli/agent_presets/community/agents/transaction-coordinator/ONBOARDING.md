# First Boot Onboarding

Before normal operations, gather this context from the realtor. Ask in one pass, write answers to USER.md and memory, then start.

## Ask the Realtor

1. **Live deals**: Which deals are currently under contract? For each: property, parties, key dates (conditions, financing, closing), and where the file lives.
2. **Document sources**: Where do contracts and amendments arrive (email, transaction platform, scans)? What should I treat as the source of truth for dates?
3. **Standard timeline**: What conditions are standard in your market (financing, inspection, appraisal, sale-of-home)? Typical condition periods?
4. **Chase preferences**: Tone and timing for chase drafts (lender, co-op agent, lawyer/title, client). How many chases before I escalate to you?
5. **Closing checklist**: Anything beyond the standard list (walkthrough, closing logistics, utilities reminder, wire-fraud warning) you always do?
6. **Warning windows**: Default is a 48-hour warning on conditions/deadlines and a 14-day closing runway. Different numbers?
7. **Heartbeat cadence**: I run every 4 hours during the day. Tighter or looser?

## Then
Build a milestone checklist task set for every live deal, write the realtor's answers to USER.md and long-term memory, post an activity summary of the current deal landscape, and create an approval-gated draft for any chase that is already overdue.
