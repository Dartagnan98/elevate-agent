# Elevate 1.2.47 — release notes

## Compaction no longer floods the chat

When a long conversation compacts, the chat no longer displays the internal
compaction summary, preserved plan/task snapshots, or routing scaffolding as
giant messages. The conversation just continues — the run keeps working
through a compaction and the answer lands as normal.

## Thinking reads as prose again

A 1.2.46 change could split streamed thinking into one-word fragments when
tool activity interleaved with it. Thinking renders as full passages again.
