# Elevate 1.2.44 — release notes

The steer bar is back, the way it should work.

## Enter queues; the bar steers

Pressing Enter while the agent is working now always places your message in
the queue — it shows up as a bar above the message box with the message text,
a **Steer now** button on the right to inject it into the running turn, and
**Cancel** to drop it. Steered messages land in the run's timeline with a
"Conversation steered" marker right where they cut in. No more guessing
where a mid-run message went.
