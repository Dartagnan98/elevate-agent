# Elevate 1.2.26 — release notes

Small follow-up to 1.2.25.

## Background tasks show their token spend

- **Live meter on running tasks.** Background-task cards in the side panel and
  the inline subagent rows in a chat turn now show "↓ N out" while the task
  runs — the same live output-token meter the thinking pill shows.
- **Real totals when finished.** Once a sub-agent completes, the estimate is
  replaced by the session's actual output-token total, so the panel doubles as
  a quick cost read on delegated work.
