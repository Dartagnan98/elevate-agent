# Elevate 1.2.50 — release notes

## Reopened chats stay clean

When reasoning display is turned off, reopening a chat no longer shows
persisted internal thinking fragments in the activity timeline. Tool activity
still appears, but private reasoning stays hidden.

## App update version restored

This release follows the local 1.2.49 dogfood build, so the public update lane
uses 1.2.50 to make sure every installed app can detect and install the fix.
