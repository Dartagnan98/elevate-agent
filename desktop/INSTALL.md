# Installing Elevate On Mac

Quick guide for first-time install. Takes about 2 minutes.

## 1. Download

Grab the `.dmg` that matches your Mac:

- **Apple Silicon** (M1, M2, M3, M4, M5): `Elevate-latest-mac-arm64.dmg`
- **Intel Mac**: `Elevate-latest-mac-x64.dmg`

Not sure which one you have? Click the Apple menu, choose About This Mac, and
check the chip line. Apple chips use arm64. Intel chips use x64.

## 2. Install

1. Double-click the DMG file you downloaded.
2. A window opens showing the Elevate icon and an Applications folder shortcut.
3. Drag **Elevate** onto the **Applications** folder.
4. Wait for the copy to finish (a few seconds).
5. Eject the disk image.

## 3. First Launch

The production app is signed with Developer ID and notarized by Apple. Open it
from Applications like any other Mac app.

1. Open Finder, then Applications.
2. Find **Elevate**.
3. Double-click **Elevate**.
4. Approve microphone or Messages prompts only when you use those features.

## 4. Updating Later

Elevate checks for signed updates from the Elevate update feed. If you install
manually, repeat steps 1 and 2 to install over the old copy. Your data stays put.

## Troubleshooting

**"Elevate is damaged and cannot be opened"** — download the DMG again from the
official Elevate link. If it still happens, contact support with a screenshot.

**App will not open at all** — send your Elevate support contact a screenshot of
what you see.
